TripUs is a decision making web application which helps you and your friends plan the best trip based on your and your friends' requirements. To start using the application you only need to enter the website, create a group page and invite all the group members into the group room. Each member of the group sets what he would and would not like to do in the trip and after everyone has finished entering his requirements TripUs will analyze everybody’s requests and recommend the best locations for the trip based on it.

How does it work?

Planning your trip has never been easier!

  • Create a group room.

    TripUs provides an easy and intuitive web interface that helps you plan your trip without any headaches. You only need to click a couple of buttons before the application plans the whole trip for you. The first step is to create a group page. Just click on the create group icon, write the group name and you will get a group code used to invite other team members to the room.

  • Invite all the team members.

    When the group room is created all you need to do is to share the code generated during the group creation with your friends. Each friend will then be able to use the code to join the group room by clicking on "Join Room" icon above and providing the code.

  • Enter your requirements.

    After all the team members have gathered in a group room you can start entering the trip requirements. Every member of the group can provide requirements of different types like different kinds of activities like ski, museums, pubs, etc., preferred country or favored dates for the trip. Of course all the requirements are optional and you do not have to enter them all.

  • Get the best destination for a trip.

    After all the restrictions are supplied the you can submit them and when all the team members have submitted their requirements the application can start analyzing it and preparing the list of recommended countries. At the end the web site will output a list of several countries that suit all the group requirements the most. Just pick the country you like most and you are ready for you perfect trip!
